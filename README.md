# Genshin-Artifact-Score-Simulator
![Genshin Artifact Score Estimator](https://user-images.githubusercontent.com/38092328/219904373-2966eca3-d92b-432d-add7-3537ef27f7a1.png)
ArtifactSim is a tool for Genshin Impact that helps players to get best artifact with powerful simulation.

## Feature
- AI auto status input
- Unique score prediction system
- JP/EN Light/Dark mode
- Powerful sort and filter
## How to use
# [Artifact Sim](https://artifact.nekodigi.com/)  
[Tutorial for Japanese](https://nekodigi.hatenablog.com/entry/2023/02/05/%E3%80%90%E5%8E%9F%E7%A5%9E%E3%80%91%E6%9C%80%E5%B0%8F%E3%81%AE%E3%83%A2%E3%83%A9%E3%81%A7%E3%80%81%E6%9C%80%E9%AB%98%E3%81%AE%E8%81%96%E9%81%BA%E7%89%A9%E3%82%92%EF%BC%81%E6%96%B0%E3%81%97%E3%81%8F)
### Step1 : Preparation for Screenshots

<img width="800" alt="Screenshot" src="https://user-images.githubusercontent.com/38092328/219904652-450b11a5-6188-4949-96fc-0afa833fd12d.png">

Prepare a screenshot of the artifact list from the Genshin Impact game screen.  
Please take a picture so that the artifact fits.  

### Step2 : Open the Artifact Editor

<img width="800" alt="Screenshot 2023-02-19 085532" src="https://user-images.githubusercontent.com/38092328/219905130-bdb807f1-6815-4128-a910-7c66c2eeae5c.png">

Open Artifact Sim and tap "Open Editor".

<img width="800"  alt="Screenshot 2023-02-19 085233" src="https://user-images.githubusercontent.com/38092328/219904710-4ada2380-9aea-46f9-85d6-45e9ddadee44.png">

After editor opens, tap "Add Artifact" to open the Artifact Editor.

### Step3 : Add Artifact

<img width="800" alt="Screenshot 2023-02-19 085717" src="https://user-images.githubusercontent.com/38092328/219904838-deda1e4b-4778-4c36-ab24-0d542f96cd0d.png">

Tap "Upload Inventory Capture" to upload image.
Please note that it may take some time for the autofill to load.

<img width="800" alt="Screenshot 2023-02-19 085849" src="https://user-images.githubusercontent.com/38092328/219904905-61f4245f-18f5-46bb-8443-8cfbc0a768c9.png">

If you do well, artifact level, stats and predicted score will be automatically updated.  
Tap "Add" to add and save.  
If it doesn't work, you can tap the correct value to fix it or enter it manually.

# Step4 : Find the Desired Artifact

<img  alt="Screenshot 2023-02-19 090259" src="https://user-images.githubusercontent.com/38092328/219905002-c60d6378-59ba-4633-b505-085ac07194d1.png">

You can filter artifacts by score, level, artifact set, and more. You can sort by predicted score to select the best artifact.
